RucStore 是一个极其简化的购物系统，用于《数据库系统概论》实验教学。

## 实验环境：
- 数据库：MySQL
- 编程语言：Python
- Web 框架：Flask
- 前端框架: Bootstrap
- 推荐编辑器：VSCode

## 项目说明
demo 实现了注册登录、商品交易等功能，按照 [配置文档](_doc/RucStore%20配置文档.md) 配置好环境之后，进入到 `src` 文件 夹下，执行命令 `python run.py` 运行 demo，默认情况下使用 **5000** 端口访问
