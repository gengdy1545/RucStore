**RucStore** 是一个极其简化的购物系统，支持**单供应商多顾客**，用于《数据库系统概论》实验教学。

## 实验环境：
- 数据库：MySQL
- 编程语言：Python
- Web 框架：Flask
- 前端框架: Bootstrap
- 推荐编辑器：VSCode
